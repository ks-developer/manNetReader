ネットリーダーⅡ 更新履歴


各バージョンの更新内容を紹介します。

Ver.2.32

2019年3月27日
  1. 特定のページを開くと動作を停止し、終了してしまう不具合が修正されました。
  2. フレーム内のリンクをクリックしても開けない不具合が修正されました。
  3. ページ内の検索でユニコード文字を検索すると、検索履歴に正しく登録されない不具合が修正されました。
  4. チェックボックスが表示されない場合がある不具合が修正されました。
  5. 画像ボタンの文字を正しく読み上げない不具合修正。
  6. その他、細かな不具合が修正されました。

Ver.2.31

2017年10月16日
  1. ページの内容が更新されても、テキスト画面に反映されない不具合が修正されました。

Ver.2.30

2017年10月2日
  1. YouTubeアドインから検索した後、YouTubeページを開くと、BackSpaceで検索結果に戻れない不具合が修正されました。
  2. Alt+K 拡張メニューに、アクションメニューを追加し、マウスクリックコマンドを追加しました。
    このコマンドは、グラフィック画面をマウスクリックしないと正常に動作しないページに対応するコマンドです。
  3. テキスト画面の更新処理を変更し、テキスト画面に何も表示されない症状を緩和しました。

Ver.2.29

2017年7月4日
  1. YouTubeアドインの再生中、Ctrl+↑↓キーで変更した音量が、しばらくすると元の音量に戻ってしまう問題が改善されました。
  2. ネットリーダーを終了してAOKメニューに移動したとき、AOKメニューから音声ガイドが流れない不具合が修正されました。
  3. ウェブページの検索にて、検索結果ページから各ページへのリンクを開いた後、BackSpaceで検索結果ページに戻った際、カーソルが元のリンクに戻らず、ページの先頭に移動している不具合が修正されました。
  4. 一部のページの内容が点字ピンディスプレイに表示されない不具合が修正されました。

Ver.2.28

2017年2月13日
  1. 内容の更新が頻繁にあるページで、動作が停止する不具合が修正されました。
  2. amazonを別のネットリーダーで開くと、環境により動作が停止する不具合が修正されました
  3. 特定のエディットに移動した際、エディットのラベルを読み上げない不具合が修正されました。
  4. 特定のページで、開くと表示された「サーバー使用中」の画面が表示されなくなりました。
  5. 高DPIディスプレイに対応となりました。

Ver.2.27

2016年9月21日
  1. 環境メニューの 環境のクリーンアップに「インターネット設定のリセット」が追加されました。
    今までは、できていたページの閲覧や入力が、できなくなった場合、このリセットで解決することがあります。
  2. リンク先を別のネットリーダーで開いた直後、および、開いたネットリーダーを閉じる際に、動作が停止する不具合が修正されました。
    この不具合は、古いPC-Talkerとの組み合わせで発生するパターンが確認されています。PC-Talkerも最新のバージョンに更新してください。

  2. リンク先を別のネットリーダーで開き、閉じた後、元のネットリーダーの動作が停止する不具合が修正されました。
  3. 不特定のページにて、ランタイムエラー R6025で、動作が停止する不具合が修正されました。

Ver.2.20

2016年5月6日
  1. アドイン機能が追加されました。
    まずは、YouTubeアドインがリリースされました。クイックアクセスページまたはアドインメニューから、YouTubeアドインの無料の体験版をお試しいただけます。
    今後、別のアドインのリリースも予定されていますのでご期待ください。
  2. クイックアクセスページが追加されました。
    クイックアクセスページは、よく利用されるページが集められたページです。
    ネットリーダーのトップに配置されたような存在で、BackSpace または Alt + ←で、前に開いたページに戻り続けると、最後に行き着くのがクイックアクセスページです。
    直接開くには、Ctrl+Q または ファイルメニューの クイックアクセスを実行します。
    また、クイックアクセスページをホームに登録して開くことも可能です。
  3. アドインメニューが追加されました。
    表示メニューの アドインメニューで 表示と非表示が切り替えられます。
  4. 自動通知 する／しない のショートカットキーが Ctrl+Q から Ctrl+W に変更されました。
  5. Windows10にて、楽天市場のページが開かない問題が解決されました。

Ver.2.02

2015年7月21日
  1. 自動通知機能はページ変化の検知精度が高められました。
  2. テキスト画面とグラフィック画面のカレットは同期しながら移動しますが、その同期精度が高まりました。
  3. FaceBookのログインボタンが表示されない不具合が修正されました。

    Internet Explorer10は、一般的に、動作が不安定な面があるようです。Internet Explorer10をお使いの場合は、Internet Explorer11へのバージョンアップをお勧めします。

  6. 既読リンクへジャンプコマンド（Vキー）で動作が停止する不具合が修正されました。
  7. その他、細かな不具合が修正されました。

Ver.2.01

2015年6月5日
  1. ウェブページの検索、検索エンジン googleでは、検索結果ページに予測検索が明記されるようになりました。
    例えば、ウェブページの検索で「ふじやま」と入力し、検索エンジンを googleにして実行すると、googleの予測により「富士山」の検索が行われています。
    この場合、検索結果ページの最初に、次の2行で予測検索であることが説明されます。
    　次の検索結果を表示しています 富士山
    　元の検索キーワード ふじやま
    もし、「ふじやま」で検索を行いたかった場合は、上記の「ふじやま」部分がリンクになっていますので、そのリンクから再検索が可能です。
  2. ウェブページの検索、検索エンジン googleの関連キーワード機能に対応しました。
    例えば、ウェブページの検索で「富士山」と入力し、検索エンジンを googleにして実行すると、検索結果ページの下端に次のようなリンクが現れます。
    　関連するキーワード
    　富士山 噴火
    　富士山 地震
    　富士山 登山
    　などなど。
    これらのリンクより再検索が可能です。
  3. スマートナビ機能 「前のナビ地点へ」Alt+↑ がジャンプしない不具合が修正されました。
  4. YouTubeを再生中、「YouTubeプレーヤー」でEnterを押すと、再生の停止ではなくて、別のページが開く不具合が修正されました。
  5. その他、細かな不具合が修正されました。
