ページ読み上げ中の操作キー

ページの読み上げ中に、内容を聞き逃してしまったり、もう一度 注意深く聞いてみたいときは読み上げ位置を戻すことができます。
逆に、今聞いている内容をスキップさせたいときは、読み上げ位置を進めることができます。
また、読み上げ中にリンクを実行したり、読み上げ位置をジャンプすることもできます。
以下に、読み上げ中の操作コマンドを紹介します。

　段落単位で読み上げ位置を移動します。
　・段落の先頭に戻る　　　　　　　←
　・次の段落先頭に進む　　　　　　↓
　・前の段落先頭に戻る　　　　　　↑

　空白や句読点で区切られた文節へ 読み上げ位置を進めます。
　・次の文節に進む　　　　　　　　→

　空白行に 読み上げ位置を移動します。
　空白行とは、よく 文と文の内容の切れ間に挿入されている、改行文字だけの行です。
　・次の空白行に進む　　　　　　　Space
　・前の空白行に戻る　　　　　　　Shift + Space

　見出し段落に、読み上げ位置を移動します。
　見出し段落とは、先頭に 1. や 1-1.などが書かれているタイトルのことです。
　・次の見出し段落に進む　　　　　Shift + ↓
　・前の見出し段落に戻る　　　　　Shift + ↑

　記号段落に 読み上げ位置を移動します。
　記号段落とは、先頭に ●や①などの記号が書かれている段落のことです。
　・次の記号段落に進む　　　　　　Ctrl + ↓
　・前の記号段落に戻る　　　　　　Ctrl + ↑

　見出し段落 または、記号段落に 読み上げ位置を移動します。
　・次の見出し段落 または 記号段落に進む　　Ctrl + Shift + ↓
　・前の見出し段落 または 記号段落に戻る　　Ctrl + Shift + ↑

　一定の行数分、読み上げ位置を移動します。
　・前の２０の倍数行へ進む　　　　PageUp
　・次の２０の倍数行へ戻る　　　　PageDown

　次のフレーム、前のフレームへ 読み上げ位置を移動します。
　・次のフレームへ進む　　　　　　Ctrl + Tab
　・前のフレームへ戻る　　　　　　Ctrl + Shift + Tab

　読み上げ中にリンクに差し掛かった場合、リンクを実行することができます。
　・リンクの実行　　　　　　　　　Enter

　読み上げ位置を インテリジェントジャンプします。
　・前の位置にインテリジェントジャンプ　　　Ctrl + PageUp
　・次の位置にインテリジェントジャンプ　　　Ctrl + PageDown

　読み上げ位置を スマートナビジャンプします。
　・次のナビ地点へ進む　　　　　　Alt + ↓
　・前のナビ地点へ戻る　　　　　　Alt + ↑

　読み上げ中に音声スピードや、音声ボリュームを切り替えできます。
　・音声スピード切替　　　　　　　F7
　・音声ボリューム切替　　　　　　F11

この他、読み上げ中には、数多くのジャンプコマンドが使用できます。
ジャンプコマンドは《 ページ操作 上級 - ジャンプ》を参照してください。

また、ページマークへのジャンプも可能です。
ページマークは《 ページ操作 上級 - ページマーク》を参照してください。
