５０オン順点字コマンド

 
点字コマンドを５０オン順に記載しました。参考にしてください。
なお、点字コマンドのない機能は記載していません。

【あの行】
あ + Space   選択項目の決定
い + Space   テーブルモード On/Off
い + 無変換  フリーカーソルモード On/Off
う + Space   ジャンプ 次のコンボボックス
う + 無変換  ジャンプ 前のコンボボックス
え + Space   ジャンプ 次のエディット
え + 無変換  ジャンプ 前のエディット
お + Space   カーソル位置から全文読み
お + 無変換  文頭から全文読み

【かの行】
か + Space   カーソル位置を行と列で読み上げ
か + 無変換  グラフィック画面の拡大
き + Space   ジャンプ 次のキーワード
き + 無変換  ジャンプ 前のキーワード
く + Space   クイックビュー
く + 無変換  画面表示の切り替え
け + Space   ウェブページの検索
こ + Space   ジャンプ 次のフォーム
こ + 無変換  ジャンプ 前のフォーム

【さの行】
さ + Space   テキスト画面の表示形式
し + Space   ジャンプ 次のプッシュボタン
し + 無変換  ジャンプ 前のプッシュボタン
す + Space   スクリーンリーダーモード On/Off
す + 無変換  スマートフォン用のページを開く
せ + Space   名前を付けて保存
せ + 無変換  段落の先頭へ

【たの行】
た + Space   タブキー
た + 無変換  シフト＋タブキー
ち + Space   ジャンプ 次のチェックボックス
ち + 無変換  ジャンプ 前のチェックボックス
つ + Space   お気に入りに追加
て + Space   ジャンプ 次のテーブル
て + 無変換  ジャンプ 前のテーブル
と + Space   ページの先頭へ

【なの行】
な + Space   ページ名読み上げ
に + 無変換  ジャンプ 次のフォーム
ね + Space   次の２０の倍数行へ
ね + 無変換  次の位置にインテリジェントジャンプ

【はの行】
は + Space   前の２０の倍数行へ
は + 無変換  前の位置にインテリジェントジャンプ
ひ + Space   ジャンプ 次の行頭日付
ひ + 無変換  ジャンプ 前の行頭日付
ふ + Space   範囲選択の開始
ふ + 無変換  印刷
へ + Space   ページマークの設定
へ + 無変換  ページマークのクリア
ほ + Space   ホームに戻る
ほ + 無変換  コピー

【まの行】
ま + Space   次のページマークにジャンプ
ま + 無変換  前のページマークにジャンプ
み + Space   ジャンプ 次の見出し
み + 無変換  ジャンプ 前の見出し
む + Space   ジャンプ 次のリンクのないテキスト
む + 無変換  ジャンプ 前のリンクのないテキスト
め + Space   ネットリーダーの終了
も + Space   ページ内の検索
も + 無変換  直前のジャンプを戻る

【やの行】
ゆ + Space   ジャンプ 次の既読リンク
ゆ + 無変換  ジャンプ 前の既読リンク
よ + Space   ジャンプ 全文読みの読み始め位置

【らの行】
ら + Space   ページの末尾へ
ら + 無変換  段落の末尾へ
り + Space   ジャンプ 次のリスト
り + 無変換  ジャンプ 前のリスト
る + Space   段落読み
れ + Space   ジャンプ 次のプレーヤー
れ + 無変換  ジャンプ 前のプレーヤー
ろ + Space   ジャンプ 次のブロック
ろ + 無変換  ジャンプ 前のブロック

【わの行】
わ + Space   一行読み
を + 無変換  すべて選択

【点の組み合わせ】
6の点 + Space       受信の中止
2 3の点 + Space     前に戻る
2 6の点 + Space     ジャンプ 次のラジオボタン
2 6の点 + 無変換    ジャンプ 前のラジオボタン
4 5の点 + Space     前を検索
4 6の点 + Space     次を検索
5 6の点 + Space     次に進む
3 5 6の点 + Space   ジャンプ 次の未読リンク
3 5 6の点 + 無変換  ジャンプ 前の未読リンク
