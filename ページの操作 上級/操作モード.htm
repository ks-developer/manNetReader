操作モード

●テーブルモード

テーブルモードは、特有のカーソルの移動方法により、テーブルの内容の理解の助けとなる機能です。
　・テーブルモード オン　Insert
　・テーブルモード オフ　Insert または Escape
　・点字では オン オフ 切り替え　い + Space

切り替えは、カーソルがテーブルにある場合にのみ有効で、オンにしたテーブルのカーソルは次のように移動します。
　・↑　上のセルに移動します。
　・↓　下のセルに移動します。
　・←　左のセルに移動します。
　・→　右のセルに移動します。
それぞれ、上下左右の端のセルを越えての移動はしません。
このようにカーソルは、セル単位で移動し、端のセルで止まるのでテーブル外に抜け出すことはありません。現在、このテーブルを集中して確認している状態と言うことで、あえて、テーブルを抜け出すことを禁止しています。
テーブル外に抜け出すには、テーブルモードをオフにし、通常のカーソル移動に戻します。

また、テーブルの内容を確認するために役立つ、次のコマンドも用意されています。
　・移動先セルの読み上げ オン／オフ Ctrl+F9
　　オンにすると、カーソルが移動したセルの説明を読み上げます。
　・セル番号の読み上げ Shift+F9
　　カーソルのあるセル番号、結合状態や枠線の情報を読み上げます。

※補足
主に、時刻表やカレンダー、番組表などの一覧表として使われたテーブルの確認に威力が発揮されます。ページのレイアウトを構成するために使われているテーブルなどでは、この機能はあまり有効ではないかもしれません。


●フリーカーソルモード

カーソルキーの移動単位を変更します。
　・フリーカーソルモード オン オフ 切り替え　Shift + Insert
　・点字では　い + 無変換

モードをオンにすると、矢印キーの移動単位が次のように変わります。
　・↑　上の行に移動します。
　・↓　下の行に移動します。
　・←　左の文字に移動します。
　・→　右の文字に移動します。
このようにカーソルは １行 １文字単位の移動にかわるので、じっくり文字単位に確認したい場面でオンにします。
また、拡大画面をお使いの場合、通常のカーソル移動では、文字が画面外にあり表示されないことがあります。この場合、モードのオンで、カーソルの移動に伴い、画面外の文字も画面内にスクロールするようになり、すべての文字が表示できます。


●スクリーンリーダーモード

スクリーンリーダーのキー操作に切り替えます。
　・スクリーンリーダーモード オン　Ctrl + Insert
　・スクリーンリーダーモード オフ　Ctrl + Insert または Escape
　・点字では オン オフ 切り替え　す + Space （スクリーンリーダーの す）

スクリーンリーダーモードをオンにする場合は、次のような場面を想定しています。
　・PDFファイルの読み上げを行う。
　・ページ内に、フラッシュのような操作できないコンテンツがある。
　・実行できないリンクやボタンがある。
この他、操作できない場面があれば、スクリーンリーダーモードをオンにしてみます。
オンなら操作が可能になる場合があります。

モードをオンにすると、Internet ExplorerをPC-Talkerを利用して読み上げることと同じ状態になり、同じキー操作になります。
Internet ExplorerをPC-Talkerと共に使ったことのあるかたならそのまま利用できるモードであると思いますが、逆に、この両ソフトの操作方法を学習されていないと利用できない上級者向けのモードでもあります。
